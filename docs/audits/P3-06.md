Generalize the remaining Standard-only MTGO publication helpers while preserving the existing Standard commands, human-review workflow, public paths, and output bytes.

## Implementation

- Added `src/mtgmeta/mtgo/pickup.py` with explicit repository root and format selection.
- Moved Weekly Pickup candidate generation, manual approval publication, known-archetype state, and Pickup catalog handling behind the validated format registry.
- Routed `weekly_pickup.py` and `gen_meta.py` through the shared implementation while preserving the legacy Standard commands and helper imports.
- Made metadata rule and output paths configuration-driven.
- Required `catalog_generation` authorization before statistics, matchup, or Pickup catalog output can be written.
- Preserved the rule that candidate generation alone never publishes data or updates known-archetype state.
