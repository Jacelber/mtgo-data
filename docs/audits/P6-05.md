## Purpose and boundary

P6-05 enables MTGO Modern event and rolling-range statistics as the next non-public Modern product layer. It does not enable Modern matchup data, Weekly Pickup, metadata, the broader format catalog, production workflow execution, public-format discovery, or front-end selection.

The P6-04A publication record is closed first: documentation commit `4a2516ed72f93931aee4eac7d6a743bb64fa7758`, pull request #83, and merge commit `46146c5503fa4cb51aead309f868f2fbb4cd648b` passed remote repository validation.

## Implemented contract

- Modern remains executable and non-public.
- Its enabled capabilities are exactly `classification`, `event_statistics`, and `range_statistics`.
- The statistics range index is part of `range_statistics`; it no longer requires the broader `catalog_generation` capability.
- Statistics read only exact embedded `CMODERN` event documents. Any cross-format document blocks generation before the output directory is created.
- Primary statistics and deck-construction baselines aggregate internally by stable parent archetype ID.
- New Modern range rows and representative-deck entries emit the parent archetype ID beside the display name.
- Selected subtypes remain diagnostic: they do not create separate statistic rows and cannot double-count a parent population.
- Existing Standard documents keep their legacy public shape and remain byte-identical.
